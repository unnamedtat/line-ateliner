# Code of Conduct

## Our Pledge

We want Line Atelier to be a respectful, welcoming project for contributors of different backgrounds and experience levels.

## Our Standards

Examples of behavior that helps the project:

- Being respectful in discussions and reviews
- Assuming good intent and asking clarifying questions
- Giving actionable, specific feedback
- Focusing criticism on code and behavior, not people

Examples of unacceptable behavior:

- Harassment, insults, or discriminatory language
- Personal attacks or intimidation
- Publishing private information without permission
- Disruptive or hostile conduct in project spaces

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that do not align with this code of conduct.

## Scope

This code of conduct applies within the repository, issue tracker, pull requests, and related community spaces.
